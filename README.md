# QR Code Generator

A Python project that converts text and URLs into QR codes.

## Features
- Generate QR code from text
- Generate QR code from URLs
- Saves QR code as an image file

## Technologies Used
- Python
- qrcode library

## How to Run
1. Install the required package:
```bash
pip install qrcode[pil]
```
